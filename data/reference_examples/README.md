# Reference Examples for MatLLMSearch

These structures serve as FORMAT REFERENCES for the LLM.
They show the POSCAR format but are NOT used as parent structures.

## Structures:

1. **NaCl** - Simple ionic compound (rock salt)
2. **CaTiO3** - Complex oxide (perovskite)  
3. **SiO2** - Covalent network (quartz)

## Purpose:

The LLM sees these examples to understand:
- POSCAR format
- Different structure types
- Lattice parameters format
- Atomic positions format

The LLM will generate ENTIRELY NEW structures, not modify these.
